# Air Pollution Analysis (SQL+EXCEL Dashboard)

## Project Overview
This project analyzes air pollution data across Indian States and Union Territories using SQL and visualizes insights through an interactive Excel Dashboard.

## Tools used
-SQL Server
-Microsoft Excel
-Pivot Tables
-Data Analysis

## Key Analysis
1. State-wise AQI Analysis
2. Top 10 pollued and cleanest areas of each State/UT
3. Pollution source contribution
4. Severity category distribution

## Key Insights
- Punjab  shows the highest pollution levels.
- Industrial Emissions and Construction Dust are major pollution sources.
- Most regions falls under Moderate pollution levels.
